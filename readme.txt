== Installation ==

1. Upload the entire `tabs-builder` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Tabs Builder Panels menu to customize this plugin.
4. Copy your publish tabs shortcode and then paste into your page or post to display it.

== Frequently Asked Questions ==

= How to add new tabs ? =

Just go to Tab builder menu , now click here "Add New" Button and then tabs panel is panel in front of you.

= Does tab work on page and post both? =

Yes, it's work.  You can display tabs everywhere on your website.


= Does this work in widget area? =

Yes,  just add "Text Widget" and then paste your tab shorcode in widget text description.

= Does this tab plugin support html code?  =

yes,  you can add HTML code in tabs description field.  

= Does this tab plugin support other plugin shortcode ?  =

Yes, Just add other plugin shortcode inside tabs description field.  you can display gallery slider, anything inside tabs.

= Does this tab plugin support nested feature?  =

Yes, you will need to just add other exist tab shortcode inside current tab description field.

= is this support in mobile devices?  =

Yes this tab plugin responsive  and work on all media devices.

= Can i apply my own styles on tabs ?  =

yes , with help  of custom css you can apply your on style on tabs.
